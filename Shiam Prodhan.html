Projects

Bizness Buddy



This is an accounting software which keeps and analyse the accounts of business. The key features are generating income statement, equity statement, balance shee, calculate tax, solvency rate and acid ratio. This project is based on C# language.

ISP Based Network Solutions



This Project is based on the local isp business which provides broadband connections or ip address to the customers. Different types of routing process are shown in this project and the network simulation was done by the Cisco Packet Tracer.

Roof Garden Irrigation and Drainage using Micro controller



This project is about automatic gardeing on the roof or in the balcony. The micro controler and sensors are used to collect information of the soil and if the soil is dry then it sends the data to the micro controller and the micro controller turns on the motor and when the water reach at a certain level the motor automatically goes off and also if the water level overflows it drains the extra water outside the pot.

Tutors Point



In this project teachers and students both search and find tuitions according to their preference and locations. Tutors point is basically an app which connects both teachers and students.

Railway Management System



This project is based on database and it helps both the railway authority and the passengers to share or view different information of different trains such as location, ticket price, timing, date and time, advance booking etc.

Home Automation



In this project the users can control their home electrical appliances with their voic or phone command. The micro controller and the node mcu is used to keep connected the user with the devices over the wifi and control them.
